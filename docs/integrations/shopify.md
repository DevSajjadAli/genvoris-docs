# Shopify (coming soon)

The Shopify app is in active development. It will let you:

- Install Genvoris in two clicks from the Shopify App Store.
- Map a Genvoris plan to any Shopify product / variant.
- Auto-mint sessions for logged-in Shopify customers — no theme.liquid edits beyond a single block.
- Pause / cancel customers automatically when their Shopify subscription changes (powered by `Shop.subscriptions/*` webhooks).

Until then, the [custom integration](./custom) recipe works on Shopify too — call our REST API from your Shopify Functions / app proxy.

> Want early access? Email **integrations@genvoris.org**.
