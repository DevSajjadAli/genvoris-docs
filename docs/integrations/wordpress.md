# WordPress / WooCommerce (coming soon)

The Genvoris plugin for WordPress + WooCommerce is in development. Expected features:

- WordPress.org plugin install.
- Settings page to paste your store API key + map plan ↔ Woo subscription product.
- Automatic webhook handling for `woocommerce_subscription_status_*` events.
- A Gutenberg block + shortcode (`[genvoris_tryon]`) for product templates.

Until then, see the [custom integration](./custom) page — the REST API works the same on WordPress.

> Want early access? Email **integrations@genvoris.org**.
